Shareable Birthday Card

"Shareable Birthday Card" is a single webpage base application created by HTML, CSS, and Javascript with jQuery plugin. All images I used are SVG data and I control them by CSS animation and jQuery. There are three basic interactive functions on this website. First, users can control the position of the drink with drag-and-drop. Secondly, when users tap the balloon, it goes away with an animation. Finally, the note illustration is also interactive. When users tap or click it, the jQuery connects to JSON based iTunes database by Ajax, which can perform an asynchronous HTTP request. When the Ajax bring the iTunes data back, JavaScript Audio API receives the information and plays musics randomly.
