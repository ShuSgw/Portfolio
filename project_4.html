Chat Together

Chat Together is a web application that users can post thier messages. Also, it has a login and register system that saves users information. The purpose of the project is to show my skills in HTML, Sass, PHP, and MySQL. I used Code-kit which is one of tools that helps me to complete meta languages such as SASS. What I struggled was this was my first challenge to create web application with PHP and Mysql, so that it took time to understand how they worked. However, I could get familiar with backend languages through this project.
